# hello-hyperdb
Simple example of creating a Hyperdb backed by RocksDB and Hyperbee.

## Usage
Create database schema
```js
node schema.mjs
```
Run example of a local database backed by RocksDB

```js 
node index.mjs
```

Run example of creating a Hyperdb backed by Hyperbee (P2P)

```js 
node hyperbee.mjs
```
